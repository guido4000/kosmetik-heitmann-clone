# Kosmetik & Pflege Ingrid Heitmann - Website

A clean, modern website for a Swiss cosmetics and wellness salon located in Happerswil.

## Overview

This is a static HTML/CSS website built as a clean rewrite of the original Jimdo-based site. The focus is on elegant design, fast performance, and easy maintenance.

## Features

- Responsive design for all devices
- Clean, semantic HTML5
- Modern CSS with custom properties
- Mobile-friendly navigation
- Animated service cards with SVG icons
- Image gallery with hover effects
- Contact form
- Cookie settings page
- Legal pages (Impressum, Datenschutz)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage with hero, intro, gallery, and service cards |
| `leistungen.html` | Services page with categorized treatment offerings |
| `produkte.html` | Products page showcasing cosmetic products |
| `ueber-mich.html` | About page introducing Ingrid Heitmann |
| `kontakt.html` | Contact page with form and location info |
| `impressum.html` | Legal imprint (Swiss law) |
| `datenschutz.html` | Privacy policy |
| `cookie-einstellungen.html` | Cookie preferences management |

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, Flexbox, Grid
- **JavaScript** - Vanilla JS for mobile menu
- **Fonts** - Google Fonts (Raleway, Roboto)

## Local Development

To run locally with live reload:

```bash
npx live-server
```

Or with a simple server:

```bash
npx serve
```
